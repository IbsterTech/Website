About Broforce

Broforce is a side-scrolling run-and-gun platform video game developed by Free Lives and published by Devolver Digital.
Development began in April 2012 as a game jam entry and continued with developer and popular support, existing as an Early Access game. Wikipedia

Installing Wine!

Make Sure Your System Is UpTo Date:

sudo apt update -y && sudo apt upgrade -y

If your system is 64 bit, enable 32 bit architecture:

sudo dpkg --add-architecture i386

Download and add the repository key:

sudo mkdir -pm755 /etc/apt/keyrings && sudo wget -O /etc/apt/keyrings/winehq-archive.key https://dl.winehq.org/wine-builds/winehq.key

Wine Installation

Works for (Ubuntu 22.04, Linux Mint 21.x, etc.)

sudo wget -NP /etc/apt/sources.list.d/ https://dl.winehq.org/wine-builds/ubuntu/dists/jammy/winehq-jammy.sources

To install on (Debian, Fedora, Ubuntu(Newer Versions) etc.) other linux distros click here!

Update the package information:

sudo apt update -y && sudo apt upgrade -y

Install one of the following packages.

Stable branch

sudo apt install --install-recommends winehq-stable

Development branch

sudo apt install --install-recommends winehq-devel

Staging branch

sudo apt install --install-recommends winehq-staging

Setting Up Wine
