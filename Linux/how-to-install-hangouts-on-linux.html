About Google Hangouts

Google Hangouts was a cross-platform instant messaging service developed by Google.
It originally was a feature of Google+, becoming a standalone product in 2013, when Google also began integrating features from Google+ Messenger and Google Talk into Hangouts. Wikipedia

Make Sure Your System Is UpTo Date:

sudo apt update -y && sudo apt upgrade -y

Installation of essential dependencies:

sudo apt install snapd && sudo snap install core

Works For Other Linux Distros:

Installing Ferdi (Which will be used to install hangouts):

sudo snap install ferdi
