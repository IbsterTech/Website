About Microsoft Edge

Microsoft Edge is a proprietary, cross-platform web browser created by Microsoft.
Released in 2015 as part of Windows 10 and Xbox One, it was initially built with Microsoft's own proprietary browser engine, EdgeHTML, and their Chakra JavaScript engine. Wikipedia
