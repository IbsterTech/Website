About Minecraft :

Minecraft is a sandbox video game developed by the Swedish video game developer Mojang Studios.
The game was created by Markus "Notch" Persson in the Java programming language. Wikipedia

Installing Minecraft:

Make Sure Your System Is UpTo Date:

sudo apt update && sudo apt upgrade

Installing jdk And jre:

sudo apt install default-jre && sudo apt install default-jdk

Installing "openjdk-8-jre" And "openjdk-8-jdk"

sudo apt install openjdk-8-jre && sudo apt install openjdk-8-jdk

Verifying if java installed successfully:

java --version
