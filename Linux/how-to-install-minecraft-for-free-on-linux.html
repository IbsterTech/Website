How To Install Minecraft For Free On Linux - IbsterTech

⭐ Note : Make sure you watch the video below, to install Minecraft without any problems!

About Minecraft :

Minecraft is a sandbox video game developed by the Swedish video game developer Mojang Studios.
The game was created by Markus "Notch" Persson in the Java programming language.Wikipedia

Installing Minecraft:

Make Sure Your System Is UpTo Date:

sudo apt update && sudo apt upgrade

Installing Jdk And Jre:

sudo apt install default-jre && sudo apt install default-jdk

Installing "openjdk-8-jre" And "openjdk-8-jdk"

sudo apt install openjdk-8-jre && sudo apt install openjdk-8-jdk

Verifying if java installed successfully:

java --version

Downloading Java 8 For RPM Based Linux Distros:
https://www.oracle.com/java/technologies/javase-jre8-downloads.html
https://www.oracle.com/java/technologies/javase/javase-jdk8-downloads.html

Go To The Following Link And Download: Tlauncher

Running Minecraft For The First Time:

After you've watched and followed the tutorial, Minecraft(tlauncher) should be successfully installed.
Once your done, you can finally start using Minecraft.
Note: Sometimes the application wont run, hence Logging out and back in again or restarting your computer will usually cause minecraft(tlauncher) to run..
