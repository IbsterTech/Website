About Odoo

Odoo is a suite of business management software tools including, for example,
CRM, e-commerce, billing, accounting, manufacturing, warehouse, project management, and inventory management. Wikipedia

Make Sure Your System Is UpTo Date:

sudo apt update -y && sudo apt upgrade -y

Download the script:

sudo wget https://raw.githubusercontent.com/Yenthe666/InstallScript/16.0/odoo_install.sh

Make the script executable:

sudo chmod +x odoo_install.sh

Execute the script:

sudo ./odoo_install.sh

Accessing Odoo On Your Browser:

In order to access Odoo, browse: http://localhost:8069/ on your favorite browser.

Running Odoo For The First Time

After you've installed Odoo, a shortcut icon should be present in the application launcher of your desktop environment.
Simply search for Odoo using its search function or look through the applications list. After clicking the icon, Odoo should open up.
Sometimes the Installation does not update properly, hence Odoo won't be present in your system ..
In that case logging out and back in again or restarting your computer will usually cause your system to update and Odoo to appear in your system ..

Uninstalling Odoo - How To Uninstall Odoo From You Linux System!

For whatever reason you want uninstall odoo we got you covered!

Firstly remove the “/opt/odoo” directory:

sudo rm -R /opt/odoo

Removing the Odoo server configuration file “odoo-server.conf”:

sudo rm -f /etc/odoo-server.conf

Remove the Odoo configuration file “odoo.conf” that is utilized for configuring the database:

sudo rm -f /etc/odoo.conf

Finally, remove the “/etc/init.d/odoo-server” directory that assists in starting odoo service at system boot:

sudo rm -f /etc/init.d/odoo-server

You can also remove the dependencies that you installed whilst installing odoo:
