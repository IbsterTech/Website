About Spotify

Spotify is a Swedish Audio streaming and media services provider founded on 23 April 2006 by Daniel Ek and Martin Lorentzon.
It is the world's largest music streaming service provider, with over 381 million monthly active users, including 172 million paying subscribers, as of September 2021. Wikipedia

To Install Snap(Snap is only required for Method 1) On Other Linux Distros, Go To The Offical Snap Website Documentation : https://snapcraft.io/docs/installing-snapd

Commands(Method 1) :

sudo apt update

sudo apt install snapd

sudo snap install core

sudo systemctl restart snapd

sudo snap install spotify

Commands(Method 2) :

curl -sS https://download.spotify.com/debian/pubkey_5E3C45D7B312C643.gpg | sudo apt-key add -

echo "deb http://repository.spotify.com stable non-free" | sudo tee /etc/apt/sources.list.d/spotify.list

sudo apt-get update && sudo apt-get install spotify-client

Running Spotify For The First Time : 


After you've installed Spotify, a shortcut icon should be present in the application launcher of your desktop environment. 

Simply search for Spotify using its search function or look through the applications list. After clicking the icon, Spotify should open up.


 Note : Sometimes the application wont show in the menu, to fix this issue :  
   
    Logging out and back in again or restarting your computer will usually cause Spotify to appear.
