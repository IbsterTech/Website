About Vlc :


VLC media player is a free and open-source, portable, cross-platform media player software and streaming media server developed by the VideoLAN project.
VLC is available for desktop operating systems and mobile platforms, such as Android, iOS and iPadOS. Wikipedia 

⭐ Note : As demonstrated in the video above, there are 2 methods, Method 1 Supports More Linux Distros, Whereas Method 2 Supports Linux Distros(Debian/Ubuntu Based)..

To Install Snap(Snap is only required for Method 1) On Other Linux Distros, Go To The Offical Snap Website Documentation : https://snapcraft.io/docs/installing-snapd

Commands(Method 1) :

sudo apt update

sudo apt install snapd

sudo snap install core

sudo systemctl restart snapd

sudo snap install vlc

Commands(Method 2) :

sudo apt update

sudo apt install vlc

Running Vlc For The First Time : 


After you've installed Vlc, a shortcut icon should be present in the application launcher of your desktop environment. 

Simply search for Vlc using its search function or look through the applications list. After clicking the icon, Vlc should open up.


⭐ Note : Sometimes the application wont show in the menu, to fix this issue :  
   
    Logging out and back in again or restarting your computer will usually cause vlc to appear.
