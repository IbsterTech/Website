About WhatsApp

WhatsApp is a freeware, cross-platform, centralized instant messaging and voice-over-IP service owned by United States tech conglomerate Meta Platforms. It allows users to send text, voice messages and video messages, make voice and video calls, and share images, documents, user locations, and other content. Wikipedia

Make Sure Your System Is UpTo Date

sudo apt update -y && sudo apt upgrade -y

Installation of essential dependencies

sudo apt install snapd && sudo snap install core

Setting Up Snap

sudo systemctl restart snapd && sudo systemctl enable --now snapd apparmor

Installing Snap Documentation: https://snapcraft.io/docs/installing-snapd

Installing walc(Unofficial WhatsApp Client)

sudo snap install walc
